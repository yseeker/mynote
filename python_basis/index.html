    関数¶

    argument は値のこと、parameterは引数のこと
    positional paremetersはデフォルトなし。
    keyword parametersはデフォルトあり。
    argsは不特定多数の引数を受け取る。
    kwrdsはkeyとvalueを入れる。
    def get_average(args):
    return args
    Pythonでは全て参照渡し。
    グローバル変数、ローカル変数
    モジュールはpythonのスクリプトファイル
    関数の中でglobalを定義すると関数の中からグローバル変数を変更できる。
    constant variable 大文字で書く
    関数の中の関数：nested function (global の代わりにnon localを使う。)
    カプセル化：外からアクセスできないようにする。
    関数もオブジェクト、関数を引数でとれる、関数もreturnできる（オブジェクトとして返す。）
    Closure:状態をキープした関数。（状態を動的・静的）
    デコレータ：関数に機能を付け加える。デコレータを使う場合はargs    と**kwdsを使う。
    ラムダ関数（関数名が無い関数）、closureで書くときによく使う（戻り値、引数を使う）、filter 関数の際に使う。
    モジュールは.pyファイル 複数束ねたものを1つのフォルダにまとめたものパッケージ、それらのフォルダをまとめたものをライブラリ
    関数だけをimportするにはfrom import
    パッケージの中のモジュールのときはpackage.moduleとする。
    アンダースコアの関数を外から使えない。
    使いたいモジュールはsys pathに入れる。
    pipを使うとsite-packagesの中で管理される。

    re:正規表現
    print(re.search('[0-9]', string))
    print(re.search('^[0-9]', string)):最初の文字
    print(re.search('^[0-9]{4}', string)):最初の文字 リピート

    print(re.search('^[0-9]{2-4}', string)):最初の文字2-4文字 リピート
    print(re.search('^[0-9]{2-4}$', string)):最後の文字2-4文字 リピート

    print(re.search('a*b', 'aaaab'))左のパターンを0回以上繰り返す
    print(re.search('a+b', 'aaaab'))左のパターンを1回以上繰り返す

    print(re.search('ab?c', 'aaaab'))左のパターンを0回か1回繰り返す
